# Tarea 3 Angular 

Este proyecto permite visualizar juegos asi como editar y borrarlos.

## Como ejecutar este proyecto?

-   [ ]  En la carpeta del client:

```
npm i 
```

-   [ ] Luego ejecuta
```
npm start
```

-   [ ] Iniciar el servidor en la carpeta server

```
npm i 
```

-   [ ]  Iniciar el servidor desde su carpeta
```
npm start 
```
